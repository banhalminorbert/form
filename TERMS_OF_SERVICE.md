## 3. Nem-Garantált Jelleg

### 3.1 Az Audit Korlátai

Az audit **nem garantálja:**

```
❌ Keresési rangsorbeli javulást
❌ Internetes forgalom növekedését
❌ Szociális média követőszám emelkedését
❌ Bejelentkezéseket vagy ajánlásokat
❌ Nyilvánosságot vagy sajtó-szerzéseket
```

**Miért?**
- A Google algoritmus folyamatosan változik
- A keresési rangsor sok tényezőn múlik
- Harmadik felek (sajtó, média) döntik el az említéseket
- Az AI-válaszmotorok még fejlődni fognak

### 3.2 Felelősség Korlátozás

A Metallagentur nem vállal felelősséget az alábbiak miatt:

| Kockázat | Magyarázat | Korlátozás |
|---|---|---|
| **Harmadik féltől származó adatok** | Wikipedia, LinkedIn, IMDb pontatlansága | Metallagentur nem felelős |
| **Már meglévő online reputáció** | Régi bulvárcikkek, negatív bejegyzések | Nem törölhetőek garantáltan |
| **Algoritmus-változások** | Google, ChatGPT, Perplexity frissítése | Előre nem jósolható |
| **Jogsértő tartalom** | Az ügyfél által feltöltött szöveg, kép, videó | Az ügyfél felelőssége |
| **Szerzői jogi megsértés** | A kérdőívben feltöltött szöveg | Az ügyfél felelőssége |

### 3.3 Limitált Garancia

A Metallagentur **garantálja**, hogy:

✓ Az audit-kérdőív működik  
✓ Az audit-dokumentum elkészül  
✓ Az adatok bizalmasan kezelkednek  
✓ Az GDPR-nak megfelelően járunk el  

De:
⚠️ Nem garantálja az audit eredményeit  
⚠️ Nem felelős az internetes pozicionálás sikerességéért

## 4. Ügyfél Felelősségei

Az ügyfél feladata:

### 4.1 Pontosság

- Az audit-kérdőívben **igaz és teljes információt** adnia
- Csak **ellenőrzött tényeket** felvenni
- Hamis vagy megtévesztő adatokat **nem használni**

**Ha hazugságot fedezünk fel:**
- Az audit törlésre kerül
- Költségvetés visszatérítés nélkül
- Lehetséges jogi lépések

### 4.2 Jogszabályoknak való Megfelelés

Az ügyfél feladata biztosítani, hogy:

- Az audit-dokumentum **nem sért szerzői jogokat**
- Az audit **nem tartalmaz jogellenes tartalmat**
- Az audit **nem megsértette mások jogait** (becsület, magánszféra)

### 4.3 Szellemi Tulajdon Tisztázása

Ha az audit-kérdőívben **szerzői jogok által védett szöveget vagy képet** használ:

```
⚠️ Az ügyfél feladata elérni az engedélyt!
```

Például:
- Nagyobb szerzői jogi szerzők idézete ✓ (megfelelő attribúcióval)
- Szerzői jog által védett kép másolása ✗ (jogsértés)
- Wikipedia-szöveg közvetlen másolata ✗ (jogsértés)

## 5. Szerződéses Feltételek

### 5.1 Az Audit Megkötése

**A kérdőív kitöltése = az audit megkötése.**

1. Ügyfél kitölti az online formát
2. Metallagentur megkapja az e-mailben
3. Metallagentur elkészíti az audit-dokumentumot (15-20 munkanap)
4. Metallagentur elküldi az audit-dokumentumot

### 5.2 Fizetés és Költségvetés

**Az audit költsége:** Keresése előtt lehet egyeztetni.

```
Típus: Szabad vagy Prémium csomag
Alap audit: Ingyenes (e-mailben)
Prémium javaslatok: EUR 1.500–5.000 (egyedi szerződés)
```

### 5.3 Szerződés Időtartama

```
Audit elkészítése: 15-20 munkanap
Adatok tárolása: 30 nap (vagy szerződésig)
Dokumentum tulajdoni: Ügyfél örökös права
Felelősség: 2 év után vége
```

### 5.4 Szerződés Felmondása

**Ügyfél felmondhatja az audit-szerződést, ha:**

- Az audit még nem kezdődött el (teljes visszatérítés)
- Az audit már elkezdődött (50% visszatérítés)
- Az audit befejezése után (visszatérítés nélkül)

**Metallagentur felmondhatja, ha:**

- Az ügyfél hamis információ adott
- Az ügyfél nem fizeti ki az költségvetést
- Az ügyfél szellemi jogokat sért
